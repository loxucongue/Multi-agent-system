# 旅游路线智能顾问 — 验收清单

## 一、功能验收矩阵

| 场景 | 正常流程 | 异常流程 | 降级处理 | 状态 |
|------|---------|---------|---------|------|
| 路线推荐 | 槽位收集→KB搜索→TopK→DB详情→生成回答 | 无匹配→热门路线 | LLM超时→关键词fallback | ☐ |
| 追问详情 | 选中线路→DB查字段→生成回答 | 无active_route→提示选择 | DB查询失败→提示稍后 | ☐ |
| 签证咨询 | Coze知识库→结构化回答+风险提示 | 无匹配国家→通用建议 | Coze不可用→提示 | ☐ |
| 价格团期 | DB查询+缓存→价格区间+更新时间 | 无定价→提示联系顾问 | Redis不可用→直查DB | ☐ |
| 外围信息 | Coze工作流→天气/航班/交通 | 无城市→提示补充 | 工作流失败→提示 | ☐ |
| 重匹配 | 排除已推荐→重新搜索 | 全部排除→清空重来 | 同推荐降级 | ☐ |
| 对比方案 | 批量查DB→结构化对比表 | <2条→提示先推荐 | 同追问降级 | ☐ |
| 闲聊 | 友好回复+引导回业务 | - | LLM超时→默认回复 | ☐ |
| 留资 | 手机号验证→DB持久化→状态更新 | 格式错误→422 | 重复→409 | ☐ |

| 项目 | 标准 | 状态 |
|------|------|------|
| SSE 首事件延迟 | < 5s | ☐ |
| 完整响应时间 | < 20s | ☐ |
| 并发支持 | ≥ 10 session 同时处理 | ☐ |
| Session 过期 | 7 天自动失效 | ☐ |
| 敏感信息脱敏 | 手机号 138****1234 格式 | ☐ |
| 全链路追踪 | trace_id 贯穿请求-graph-audit | ☐ |
| SQL 注入防护 | 参数化查询 | ☐ |
